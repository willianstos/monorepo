# Shared Skills

Workspace-curated skills for reuse across Codex, Claude, and compatible workflows. Capability assets only — no repository policy or workflow authority.

## Selection

- Select by category before opening any `SKILL.md`.
- Load one relevant skill when possible.
- Do not bulk-load the full skill tree.
- Treat `.agent/catalogs/` as reference material, not the default active skill set.

## Authoring

- Keep skills explicit and operational.
- Link to repository scripts, docs, and workflows instead of embedding large copied instructions.
- Keep token cost low: short summary first, full skill only when needed.
