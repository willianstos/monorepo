# Shared Skills

> Last Updated: 2026-03-06

This directory contains workspace-curated skills intended for reuse across Codex, Claude-compatible workflows, and Antigravity when the skill format fits.

## Selection Rules

- Prefer category selection before opening any `SKILL.md`.
- Load one relevant skill when possible.
- Do not bulk-load the full skill tree.
- Treat `.agent/catalogs/` as reference material, not the default active skill set.

## Authoring Rules

- Keep skills explicit and operational.
- Prefer linking to repository scripts, docs, and workflows instead of embedding large copied instructions.
- Keep token cost low: short summary first, full skill only when needed.
