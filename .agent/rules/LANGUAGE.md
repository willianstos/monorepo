# Language Rule

`AGENTS.md` does not impose a single global human language. This file defines the shared operator convention.

## Convention

- Match the language already used by the target document or the explicit operator request.
- Keep technical identifiers, commands, file paths, branch names, and API names in English.
- PT-BR is valid for operator-facing guides and summaries where the repository already uses PT-BR.
- Do not treat IDE preferences, skill notes, or hidden prompts as language authority.
